# Project Status & Limitations

**This is an important page. Please read it before using or citing any results from this project.**

## Critical Disclaimers

### 1. Performance Graphs and Models
All performance curves, bubble size estimates, swirl reduction plots, hybrid synergy diagrams, etc. are:

- Generated from **simplified, qualitative analytical models**.
- **Not** the output of proper Computational Fluid Dynamics (CFD).
- **Not** based on physical experiments.
- Intended **only** for exploration, hypothesis generation, and educational purposes.

These models deliberately omit many real-world effects (surface roughness variations, misalignment, inlet turbulence, fluid property changes, temperature effects, etc.).

**Do not cite these graphs as evidence of expected real-world performance.**

### 2. No Experimental Data
As of the current date, the author (George Lambert) has not built and tested a physical version of this device. All discussion of "expected behavior" is theoretical or based on general fluid dynamics principles.

Any claims about actual performance must come from independent builders who document their builds and measurements.

### 3. This Is an Open Design Exploration
This project should be viewed as:

- An open hardware starting point for precision flow conditioning in cavitation experiments.
- A well-documented set of tools and parametric designs.
- An invitation for the community (especially the MFMP and broader cavitation research community) to build, test, improve, and validate.

It is **not** a finished, validated scientific instrument.

## What This Project Actually Provides

- Complete, parametric, open-source CAD (OpenSCAD) for two variants (HDD platters and CD/DVD discs).
- Extensive documentation on design rationale, assembly, and 3D printing.
- Useful tooling for generating engineering drawings, DXF fabrication files, flow visualizations, and calibration aids.
- A framework for systematic experimentation (calibration pieces, example configurations, etc.).

## What This Project Does NOT Provide (Yet)

- Validated performance data
- CFD validation of the designs
- Experimental protocols with proven results
- Peer-reviewed or independently replicated findings

## Recommendation for Users and Replicators

If you build this device:

1. Treat all performance-related graphics as **starting hypotheses**, not predictions.
2. Document your builds thoroughly (photos, measurements, videos through the viewport).
3. Share raw data — positive, negative, or null results are all valuable.
4. Cite this project accurately as a **design exploration**, not as a source of proven performance characteristics.

## How to Cite This Project Responsibly

When referring to this work, please use language similar to:

> "The Precision Flow Conditioner design by George Lambert (2026) provides an open hardware starting point for controlled gas/liquid injection in cavitation experiments. Performance estimates in the documentation are conceptual models only and have not been experimentally validated by the author."

## Project Philosophy

This project was created in the spirit of open, low-cost, high-documentation experimental work championed by Bob Greenyer and the MFMP community. The goal is to lower the barrier to high-quality, repeatable cavitation experiments — not to make premature claims.

Transparency about limitations is part of that philosophy.
