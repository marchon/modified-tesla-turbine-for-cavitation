# Rapid Prototyping Configuration (CD/DVD)

**Intent**: Fastest possible build-test cycles using cheap, easy-to-source CD/DVD discs.

**Key Changes**:
- Only 12 discs (much shorter and faster to print)
- Slightly larger gap (1.4 mm) — more forgiving on printer tolerances
- Slightly larger HVAT diameter for real-world CD hole variation

**Best For**:
- Early-stage tuning of nozzle angles and HVAT brake behavior
- Testing different inlet manifold designs
- Educational demos where you want to build multiple variants quickly

**Recommendation**: Use this config while you dial in your printer on the calibration pieces. Once happy, move to one of the higher-performance configs (or the main HDD version) for serious data collection.
